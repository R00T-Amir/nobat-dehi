# نرم‌افزار نوبت‌دهی (Queue Ticket App)

نرم‌افزار ساده‌ی نوبت‌دهی با محیط کاربری گرافیکی، مناسب اتصال به
فیش‌پرینترهای حرارتی مثل **Meva TP-1000**.

## امکانات
- ✏️ **سربرگ قابل تغییر** — متن بالای فیش (چند خطی) از داخل برنامه ویرایش می‌شود.
- 🔢 **بازه‌ی شماره‌ی قابل تنظیم** — شماره‌ی شروع و پایان دلخواه.
- 🔠 **اندازه‌ی فونت قابل تنظیم** — هم برای سربرگ و هم برای شماره‌ی نوبت (۸ سایز).
- 💾 **ذخیره‌ی خودکار تنظیمات** — در فایل `config.json`.
- 🖨️ **چاپ مستقیم روی فیش‌پرینتر** — با استفاده از دستورات ESC/POS.
- 🔁 در صورت رسیدن به آخر بازه، برنامه از شما می‌پرسد که از ابتدا شروع شود یا نه.

## پیش‌نیازها
- ویندوز (برای چاپ نیاز به `pywin32` است؛ رابط کاربری روی هر سیستم‌عاملی اجرا می‌شود ولی چاپ فقط روی ویندوز کار می‌کند)
- Python 3.8 یا بالاتر

## نصب

```bash
git clone <آدرس-ریپازیتوری-شما>
cd nobat-dehi
pip install -r requirements.txt
```

## اجرا

```bash
python app.py
```

## تنظیمات
با کلیک روی دکمه‌ی «⚙ تنظیمات» در بالای برنامه می‌توانید:
- متن سربرگ فیش را تغییر دهید
- شماره‌ی شروع و پایان بازه را مشخص کنید
- اندازه‌ی فونت سربرگ و شماره را تنظیم کنید
- نمایش یا عدم نمایش تاریخ روی فیش را انتخاب کنید
- نام دقیق پرینتر نصب‌شده در ویندوز را انتخاب یا وارد کنید

همه‌ی این تنظیمات در فایل `config.json` ذخیره می‌شوند و در اجرای بعدی
برنامه به‌طور خودکار بارگذاری می‌شوند.

## نکته درباره‌ی نام پرینتر
نام پرینتر باید دقیقاً همان چیزی باشد که در
`Control Panel > Devices and Printers` ویندوز شما ثبت شده است.
اگر نام درست نباشد، هنگام چاپ لیست پرینترهای موجود روی سیستم به شما
نمایش داده می‌شود تا نام صحیح را انتخاب کنید.

## ساخت خودکار فایل EXE (GitHub Actions)
این پروژه یک ورک‌فلوی آماده در مسیر `.github/workflows/build.yml` دارد که
با هر `push` به شاخه‌ی `main`، به‌صورت خودکار روی یک ویندوز مجازی اجرا
می‌شود و برنامه را با PyInstaller به فایل `NobatDehi.exe` تبدیل می‌کند.

### نحوه‌ی دانلود فایل exe ساخته‌شده
1. در صفحه‌ی ریپازیتوری خودتان در گیت‌هاب، به تب **Actions** بروید.
2. آخرین اجرای موفق (✅) با نام "Build EXE" را باز کنید.
3. در پایین صفحه، بخش **Artifacts** را ببینید — فایل
   `NobatDehi-Windows` را دانلود کنید (یک zip حاوی exe است).

### اجرای دستی ساخت (بدون نیاز به push)
در تب Actions، روی ورک‌فلوی "Build EXE" کلیک کنید و دکمه‌ی
**Run workflow** را بزنید.

> نکته: چون این برنامه در نهایت باید روی سیستمی که پرینتر
> Meva TP-1000 به آن وصل است اجرا شود، فایل exe را روی همان
> کامپیوتر اجرا کنید.
